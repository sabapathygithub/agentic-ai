# Agentic AI

A collection of Jupyter notebooks exploring agentic AI concepts and implementations using OpenAI and Ollama.

## Overview

This repository contains hands-on examples and experiments for learning about agentic AI systems, including agent design patterns, tool integration, and autonomous decision-making.

## Contents

- Jupyter notebooks demonstrating agentic AI concepts
- OpenAI API integration examples
- Ollama local model implementations
- Tool use and function calling patterns

## Requirements

- Python 3.8+
- Jupyter Notebook
- OpenAI API key (for OpenAI examples)
- Ollama (for local model examples)

## Getting Started

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up your OpenAI API key in your environment
4. Open and run notebooks in Jupyter

## Tools & Libraries

- OpenAI Python SDK
- Ollama
- Jupyter
